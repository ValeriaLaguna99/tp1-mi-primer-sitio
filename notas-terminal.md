1. pwd - Muestra el directorio actual.
2. ls - Lista los archivos y carpetas en el directorio actual.
3. cd nombre_carpeta - Cambia al directorio especificado.
4. mkdir nombre_carpeta - Crea una nueva carpeta.
5. touch nombre_archivo - Crea un nuevo archivo vacío.
6. cat nombre_archivo - Muestra el contenido del archivo.
7. which git - Muestra la ruta de instalación de Git.
8. code . - Abre el directorio actual en Visual Studio Code.
9. git config --list - Ver los datos de configuración del git.
10. clear -  Limpia la pantalla de la terminal.
